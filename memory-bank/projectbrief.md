# Project Brief: Company Management System

## Project Overview
A Next.js 15 application for managing company data with comprehensive CRUD operations, built using modern web technologies and following enterprise-grade patterns.

## Core Purpose
Provide administrators with a robust, scalable interface for managing company information including listing, viewing, editing, and creating company records.

## Key Technologies
- **Framework**: Next.js 15 with App Router
- **Language**: TypeScript 5
- **Styling**: TailwindCSS 4
- **UI Components**: shadcn/ui with Radix UI primitives
- **Package Manager**: pnpm
- **Development**: Turbopack for fast development builds

## Project Scope
- Company listing with search, sorting, and pagination
- Company detail views
- Company creation and editing forms
- Admin-only access control
- Responsive design for all devices
- Accessibility compliance (WCAG 2.1 AA)

## Success Criteria
- Fast page loads (<2 seconds)
- Intuitive user experience
- Mobile-first responsive design
- Comprehensive error handling
- Type-safe development experience
- High test coverage
- Performance optimization

## Architecture Principles
- Server Components by default
- Client Components only when needed
- API-first design
- Component composition over inheritance
- Progressive enhancement
- Security-first approach
