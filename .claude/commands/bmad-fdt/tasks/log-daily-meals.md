# Log Daily Meals Task

This task guides you through capturing all food and beverages consumed during the day for nutritional tracking and analysis.

## Overview

Track every meal, snack, and drink you consume throughout the day. This data forms the foundation for nutritional analysis and health insights.

## Process

### Morning Setup

[[LLM: Help the user prepare for the day's tracking and remind them of their dietary goals.]]

1. **Review Today's Goals**
   - Caloric target
   - Macronutrient targets
   - Specific nutrition focus
   - Hydration goal

2. **Set Tracking Reminder**
   - Decide on logging frequency
   - Set notification times
   - Choose quick-log method

### Throughout the Day

[[LLM: Guide the user through quick, accurate food entry for each meal and snack.]]

1. **Breakfast Logging**
   - Record all items consumed
   - Note portion sizes
   - Include beverages
   - Log preparation method if relevant

2. **Mid-Morning Checkpoint**
   - Capture any snacks
   - Note hydration
   - Quick calorie check

3. **Lunch Entry**
   - Restaurant or home meal details
   - Include sauces and add-ons
   - Beverage logging
   - Portion verification

4. **Afternoon & Evening**
   - Snack logging
   - Dinner details
   - Evening beverages
   - Special items (desserts, treats)

### End-of-Day Review

[[LLM: Provide a daily summary and ask if any meals or snacks were missed.]]

1. **Complete Entry Check**
   - Review all logged items
   - Verify portion sizes
   - Confirm meal categorization
   - Add any forgotten items

2. **Quick Summary**
   - Total calories
   - Macro breakdown
   - Hydration status
   - Adherence to goals

3. **Tomorrow's Prep**
   - Note successful aspects
   - Identify challenges
   - Plan adjustments
   - Set intention for next day

## Logging Tips

- Log immediately after eating (better accuracy)
- Use consistent portion descriptors (cups, grams, pieces)
- Include cooking oils and seasonings (they add calories)
- Note restaurant names for consistency
- Screenshot menus for reference

## Data Entry Formats

### Quick Entry
```
[Food Name] | [Portion Size] | [Meal Type] | [Time]
Example: Grilled chicken breast | 150g | Lunch | 12:30 PM
```

### Detailed Entry
```
Food: [Name]
Portion: [Size & measurement]
Preparation: [Method/notes]
Ingredients added: [Oils, sauces, etc.]
Meal time: [Breakfast/Lunch/Dinner/Snack]
Timestamp: [Date & time]
```

## Common Challenges

**Too busy to log?**
Use quick voice entry or text shorthand

**Uncertain portion size?**
Reference standard serving descriptions or use your hand as guide

**Restaurant unknown nutritional info?**
Log similar restaurant dishes or generic equivalents

**Multiple ingredients?**
Break down into component parts for accuracy

## Success Metrics

- Consistency: Logging every meal
- Accuracy: Portion size precision within 10%
- Completeness: No meals missed
- Timeliness: Logged within hours of consumption
